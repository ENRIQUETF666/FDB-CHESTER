# TIENDA ONLINE 
  ## PROYECTO DE BASE DE DATOS 

## DESCRIPCIÓN 

El proyecto consiste en diseñar e implementar una base de datos para una tienda online que vende productos electrónicos y ropa, permitiendo gestionar clientes, productos, pedidos y pagos de manera eficiente. incluye entidades principales como cliente, producto, pedido y detalle_pedido, con relaciones que simulan el flujo de compras reales.

## MOTIVACIÓN 

Elegí crear una base de datos sobre una tienda online porque siempre me ha llamado la atención cómo funciona el comercio digital detrás de lo que el cliente ve en pantalla. Más allá del diseño atractivo y los botones de compra, existe toda una estructura organizada que permite gestionar productos, clientes, pedidos y pagos de manera eficiente. 

Además, quise desarrollar este proyecto pensando en el crecimiento futuro del negocio. Una base de datos bien diseñada no solo almacena información, sino que también facilita la toma de decisiones, como identificar los productos más vendidos o analizar el comportamiento de los clientes. Crear esta base de datos fue una forma de aplicar mis conocimientos técnicos a un caso práctico que podría convertirse en un emprendimiento real.
